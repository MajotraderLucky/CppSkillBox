## 1. Аудиоплеер

### Что нужно

- Класс **`Track`**: name, date (`std::tm`), duration_seconds
- Класс **`Player`**: содержит список Track + state

Команды:
| Команда | Действие                                                              |
|---------|-----------------------------------------------------------------------|
| `play`  | запросить название → начать воспроизведение (если не играет)          |
| `pause` | поставить на паузу (если играет)                                      |
| `next`  | следующий случайный трек (shuffle)                                    |
| `stop`  | остановить (один раз — повторно ничего не делает)                     |
| `exit`  | выход                                                                 |

### Подсказки

- Без глобальных функций — всё через методы класса
- State машина: `playing` / `paused` / `stopped`
- При play выводить ВСЮ инфу о треке

## 2. Мобильный телефон

### Что нужно

- Класс **`Phone`**: адресная книга + методы
- Класс **`Contact`** (или вложенная структура): name + phone
- Опционально: класс **`PhoneNumber`** для валидации формата `+7 <10 цифр>`

Команды:
| Команда | Действие                                                              |
|---------|-----------------------------------------------------------------------|
| `add`   | запросить имя + номер → добавить в книгу                              |
| `call`  | запросить имя или номер → вывести `CALL <number>`                     |
| `sms`   | запросить имя/номер + сообщение → вывести `SMS <number>: <msg>`       |
| `exit`  | выход                                                                 |

### Подсказки

- Lookup через адресную книгу: имя → номер
- Валидация номера: `+7` + 10 цифр

---

## 3. Окно рабочего стола

### Что нужно

- Класс **`Window`**: x, y, width, height
- Класс **`Screen`** (опционально): сама виртуальная сетка

Параметры:
- Экран **80 × 50** пикселей
- width/height **не отрицательные**
- Координаты **в пределах экрана**

Команды:
| Команда   | Действие                                                              |
|-----------|-----------------------------------------------------------------------|
| `move`    | сдвинуть окно на вектор (dx, dy) → вывести новые координаты           |
| `resize`  | новые ширина и высота → вывести новый размер                          |
| `display` | вывести экран: `0` — пиксель вне окна, `1` — в окне                   |
| `close`   | закрыть окно, выход                                                   |

### Подсказки

- Класс окна с инвариантами (clamp координат к границам)
- Encapsulation: setter'ы с валидацией
