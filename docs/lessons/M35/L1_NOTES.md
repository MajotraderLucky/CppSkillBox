## Концепция CMake (краткое напоминание)

> В CMake все проекты делятся на **таргеты** (targets) — цели построения.

### CMakeLists.txt описывает:

| Что                    | Назначение                                    |
|------------------------|-----------------------------------------------|
| Targets                | Цели построения (executable, library, ...)    |
| Compile keys           | Флаги компиляции                              |
| Libraries              | Используемые библиотеки                       |
| Source files           | Все файлы проекта                             |
| Дополнительные скрипты | Логика, отличная от базовой                   |
