## Задание 3 (к уроку 4 — M35.5)

### Цель

Поработать с файловой системой, используя `std::filesystem`.

### Что нужно сделать

> `std::filesystem::recursive_directory_iterator` принимает на вход путь, который рекурсивно обходит. На каждом шаге выдаёт очередной путь — может быть каталог или файл.
>
> ```cpp
> for (auto& p : fs::recursive_directory_iterator("D:\\dir1"))
>     std::cout << p.path() << '\n';
> ```

`p.path()` возвращает `std::filesystem::path`, который позволяет:
- `is_regular_file()` — определить, файл или каталог
- `extension()` — получить расширение
- `extension().compare(".torrent")` — сравнить расширение

### Что реализовать

Создать **лямбда-функцию**, которая принимает на вход:
- Путь в файловой системе (`fs::path`)
- Искомое расширение в виде строки (`std::string`)

Возвращает **список имён файлов**, которые имеют данное расширение.

Объявление может начинаться так:
```cpp
auto recursiveGetFileNamesByExtension =
    [](std::filesystem::path path,
       const std::string extension) {
        // ...
    };
```
