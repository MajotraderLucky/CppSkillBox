## 1. Менеджер учёта времени

### Что нужно

Команды:
- **`begin`** — пользователь вводит название новой задачи. Если уже есть активная — она автоматически завершается, и стартует новая
- **`end`** — завершает текущую задачу. Если активной нет — ничего не делает
- **`status`** — выводит:
  - все завершённые задачи + время в часах
  - текущую активную задачу (если есть)
- **`exit`** — выход

### Подсказка

`std::time_t` = секунды. Для часов: `(end - start) / 3600.0`.

## 2. Напоминание о днях рождения

### Что нужно

1. Пользователь вводит DR друзей последовательно: имя + дата `YYYY/MM/DD`
2. Сигнал окончания: имя `end`
3. Программа находит **ближайший** день рождения относительно текущей даты
4. Выводит имя + дату в формате `MM/DD`
5. Если DR уже был в этом году — пропускается
6. Если DR сегодня — спец. сообщение (может быть несколько таких людей)

### Подсказка

Для каждого друга из его tm-структуры брать `tm_mon` и `tm_mday`, сравнивать
с сегодняшними. Игнорировать год — нас интересует только день/месяц (повторяется ежегодно).

## 3. Таймер

### Что нужно

1. Ввод длительности `MM:SS` через `std::get_time(&t, "%M:%S")`
2. Обратный отсчёт в цикле, каждую секунду — вывод оставшегося времени `MM:SS`
3. По истечении — `DING! DING! DING!`

### Подсказка

```cpp
std::time_t target = std::time(nullptr) + minutes * 60 + seconds;
while (std::time(nullptr) < target) {
    int left = static_cast<int>(target - std::time(nullptr));
    // вывод MM:SS, sleep 1 сек
}
```
