# M32.5 — Практическая работа (homework)

**Тема:** JSON (синтаксис + nlohmann/json) и Protobuf

## Цели

- Усвоить синтаксис языка JSON
- Научиться пользоваться библиотекой для чтения и записи JSON-данных
- Освоить синтаксис языка proto2 и компиляцию через `protoc`

## Что входит в работу

1. JSON-документ с информацией о киноленте
2. JSON-база из 5 фильмов + C++-программа поиска актёра
3. Protobuf-модель данных компании

---

## Задача 1. Информация о фильме с кинопортала

### Что нужно сделать

Запишите подробную информацию о киноленте в виде JSON-файла. Выберите фильм
с любимого портала, проанализируйте информацию и составьте JSON-словарь.

В информации должны присутствовать:
- страна и дата создания киноленты
- студия, которая вела съёмки
- автор сценария
- режиссёр
- продюсер
- информация обо всех главных действующих лицах и сыгравших их актёрах

### Что оценивается

- Валидность созданного JSON-документа

---

## Задача 2. Анализ данных о фильмах

### Что нужно сделать

Используя модель данных из задачи №1, создайте JSON-документ для **пяти**
различных кинолент. Ключи в этой JSON-базе данных должны быть **названиями
фильмов**, значения — словари с информацией о картинах.

Загрузите базу данных в C++-программу для анализа. Функционал:
- Поиск актёра по **имени** (или фамилии)
- Вывод в консоль названий фильмов, в которых он снимался, и его роли

### Что оценивается

- Валидность JSON-документа
- Корректность работы программы поиска

---

## Задача 3. Модель данных компании на Protobuf

### Что нужно сделать

Создайте модель данных для информации о предприятии.

**Обязательные поля (required):**
- год основания (целое 32-битное число)
- юридический адрес (строка)
- название (строка)

**Опциональные поля (optional):**
- род деятельности (строка)
- осуществление внешнеэкономической деятельности (булевый тип)

Опишите модель на языке **proto2** и сохраните в файл с расширением `.proto`.
На основе документа сгенерируйте C++-код через `protoc`.

### Что оценивается

- Валидность модели на proto2
- Успешная компиляция в C++ через `protoc`
