## Задание 1. Работа корзины с покупками

### Что нужно сделать

Реализуйте простую и безопасную модель корзины онлайн-магазина.

В начале — заполняете БД магазина через консоль. Каждый элемент = `{артикул, количество}`.

Корзина поддерживает:
- `add` — добавить товар указанного количества (по артикулу)
- `remove` — удалить товар указанного количества

Валидация:
- Артикул должен быть в БД магазина
- Количество не должно превышать доступное (на складе для add, в корзине для remove)

### Рекомендации

- БД и корзина — `std::map`
- Некорректный ввод → `std::invalid_argument`
- В других случаях → `std::runtime_error`

### Что оценивается

- Корректность логики работы при использовании исключений

## Задание 2. Игра по ловле рыбы

### Что нужно сделать

Игровое поле = 9 секторов условного пруда. В одном — **рыба**. В трёх — **сапоги** (распределяются случайно). Сапог и рыба не могут быть в одном секторе. Несколько сапог в одном секторе тоже нельзя.

Игрок указывает сектор для заброса удочки:
- Попал в **пустой** → повторить заброс
- Поймал **сапог** → игра завершается с отрицательным результатом
- Поймал **рыбу** → бросается специальное **пользовательское исключение** → программа завершается, выводит "успех + кол-во попыток"

### Рекомендации

- Бросать можно **произвольный тип** (даже `Boot*` или `Fish*`)
- В main: `std::srand(std::time(nullptr));`, потом `field[std::rand() % 9].fish = new Fish();`

### Что оценивается

- Корректность работы игровой логики
- Точность использования исключений

## Задание 3. Реестр данных на шаблонах

### Что нужно сделать

Класс реестра, хранящий **произвольные типы** значений по ключам **произвольного типа**. Ключи могут повторяться.

Операции:
- `add` — добавить элемент с ключом
- `remove` — удалить **все** элементы с заданным ключом
- `print` — напечатать все пары (key, value)
- `find` — найти все значения по ключу

Должен работать на типах: `int`, `double`, `std::string`.

### Рекомендации

- Контейнер — `std::vector` пар
- Пара (ключ, значение) — шаблонизированный класс/структура

### Что оценивается

- Корректность работы логики, реализованной с помощью шаблонов

## Задание 4. Среднее арифметическое в массиве

### Что нужно сделать

Шаблонная функция подсчёта среднего арифметического в массиве **скалярных** чисел произвольного типа (int, double, float, ...).

Ввод/вывод — через консоль.

### Сигнатура (из задания)

```cpp
template <typename T>
void input(T array[8]) {
    std::cout << "Fill the array (8):";
    for (int i = 0; i < 8; ++i) {
        /* ввод отдельных элементов */
    }
}
```

### Рекомендации

- Массив фиксированной длины (8)
- Функцию ввода — тоже шаблонизировать

### Что оценивается

- Корректность работы с разными типами (int, double, float)
