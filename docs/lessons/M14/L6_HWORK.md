## Задание 1. Банкетный стол

### Что нужно сделать

Банкетный стол на 12 персон, 2 ряда. Есть 2 VIP-места рядом во главе.

**Initial state — 3 двумерных массива:**

- **Столовые приборы** (вилка, ложка, нож): 3 на каждого + 1 десертная ложка для VIP
- **Тарелки**: 2 на каждого + 1 десертная для VIP
- **Стулья**: минимум 1 для каждого

**События после инициализации:**

1. На 5-е место 1-го ряда дама привела ребёнка → +1 стул
2. С 3-го места 2-го ряда украдена ложка → -1 ложка
3. Одна VIP поделилась десертной ложкой → -1 у VIP, +1 у того кто без
4. Официант забрал десертную тарелку у VIP → -1 тарелка у VIP

### Подсказка

```cpp
int packages[2][3] = {{2, 1, 7},{3, 5, 31}};  // init pattern
packages[0][0] += 1;  // тогда 2 → 3
```

### Что оценивается

- Все действия внутри массива соответствуют условию

## Задание 5. Пупырка

### Что нужно сделать

Пузырьковая плёнка `bool[12][12]` (true = целый, false = лопнутый).

**3 функции:**

1. **Init** — все true
2. **Print** — `o` для целого, `x` для лопнутого
3. **Pop region** — лопает прямоугольник по 4 координатам (x1,y1) – (x2,y2)

**Loop:**

- Print
- User вводит координаты региона
- Validate (в пределах массива)
- Pop region (для каждого `true` → `false`, печатаем "Pop!")
- Repeat пока вся плёнка не лопнута

### Подсказки

- При лопании цикл только по координатам региона, не по всей матрице
- `if (shell[i][j])` вместо `if (shell[i][j] == true)`
- Validate input → error если out of bounds

### Что оценивается

- Реализованы 3 функции
- Корректная работа лопания
